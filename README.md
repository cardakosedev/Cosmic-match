##### Cosmic Match - Kozmik eşleştirme



A 2D mobile memory/puzzle game developed with Unity.



###### Project Overview



A space-themed mobile puzzle game where players match symbols hidden behind asteroids to clear the path and reach the objective.



###### Features



Level-based progression system

Randomized grid generation

Difficulty balancing system

Star/score progression mechanics

Save/Load system using PlayerPrefs



###### Technologies



Unity

C#

PlayerPrefs

Animator
